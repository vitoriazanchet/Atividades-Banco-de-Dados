# Banco de Dados
Este trabalho consiste na apresentação das atividades das listas de 1 a 6 da **Disciplina de Banco de Dados** do curso de ***Programador Full Stack da Residência em Software 2026.1 do Serratec***.
## Cujo o objetivo é apontar os conhecimentos adquiridos, como:
* Criação de Banco de Dados `CREATE DATABASE`;
* Criação de Tabelas `CREATE TABLE`, com chave primaria `PRIMARY KEY` e suas respectivas colunas, podendo ser alimentadas por dados nulos `NULL` ou não `NOT NULL`, dados que não se repetem `UNIQUE`, fazer referencia a outra tabela `FOREIGN KEY` + `REFERENCES`, entre outros;
* Alteração de estrutura da tabela `ALTER TABLE`;
* Inserir dados em uma tabela `INSERT INTO`;
* Atualizar `UPDATE` ou Deletar `DELETE FROM`  dados em uma tabela;
* Selecionar dados para visulização `SELECT`;
* Agrupar informações de duas ou mais tabelas em uma mesma seleção `JOIN`;
* Filtrar dados de uma seleção `WHERE`;
* Ordenar dados de uma seleção `ORDER BY`;
* Agrupar dados de uma seleção `GROUP BY`;
* Contar dados de uma tabela ou seleção `COUNT`;
* Criar um índice de busca `CREATE INDEX`;
* Criar um resultado de colsulta pé-definido `CREATE VIEW`;
* Testar a performance de uma consulta `EXPLAIN ANALYZE`, dentre outros.
